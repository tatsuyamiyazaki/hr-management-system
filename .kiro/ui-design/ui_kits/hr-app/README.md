# Sunbit HR — App UI Kit

SUNBIT Inc. HR プラットフォームのメイン Web アプリケーション UI キット。5つの主要ビューをクリックスルー可能なプロトタイプとして実装。

## ビュー一覧

| View            | ドメイン              | 内容                                             |
| --------------- | --------------------- | ------------------------------------------------ |
| ダッシュボード  | dashboard             | HR_MANAGER 向け KPI + タスク + アクティビティ    |
| 360度評価       | evaluation + ai-coach | 進捗テーブル + AI 品質ゲート付き評価入力モーダル |
| 目標（OKR/MBO） | goal                  | 多階層目標 + 承認フロー                          |
| 1on1            | one-on-one            | タイムラインログ + 関連情報                      |
| スキルマップ    | skill                 | 部署 × カテゴリ ヒートマップ + 個人レーダー      |

## インタラクション

- サイドバーでビュー切替
- 評価テーブルの行クリックで AI コーチングモーダル（品質ゲート → 通過で確定ボタン有効化）
- 「サイクル確定」ボタンで確認 → トースト
- 目標の承認/差し戻しボタン

## 採用している設計システム

- `../../colors_and_type.css` のトークンを直接使用
- Lucide アイコン（CDN）
- レイアウト: 240px サイドバー + 56px ヘッダー + 最大幅1440px メインキャンバス
- shadcn/ui + Tailwind CSS 4（技術スタック準拠）のエンタープライズ向け見た目を再現

## 注意

- このキットは `.kiro/specs/hr-management-platform/design.md` の19ドメイン仕様に基づく推定 UI です。実装着手時にフィードバックを反映すること。
- ダミーデータはすべて架空。
