Sunbit HR Design System

SUNBIT Inc. 人事管理プラットフォーム
スキル管理・360度評価・目標管理・1on1・キャリア管理を一元化する日本語ファーストのエンタープライズ HR プラットフォーム向けデザインシステム。

プロダクト概要

SUNBIT Inc. は、組織課題（組織体制・人材配置・人事評価）を一元管理する HR プラットフォームを提供する。AI を活用したコーチング機能付き 360 度評価により、フィードバック文化の醸成と社員の継続的成長を支援する。

ユーザーロール

ADMIN等級マスタ・ウェイト設定・監査ログ監視・AI コスト管理
HR_MANAGER評価サイクルの設定・進捗モニタリング・総合評価の確定・異議申立ての審査
MANAGER部下の目標承認・1on1 ログ蓄積・AI 対話で高品質フィードバック入力
EMPLOYEE自己評価・ピア評価・キャリア希望表明・FB 受領・異議申立て

差別化ポイント

  • ピア主導の 360 度評価: 「誰を評価するか」を社員自身が選び、最低評価者数で匿名性と統計的信頼性を両立
  • AI 品質ゲート: 品質基準を満たすまで評価送信をブロックし、建設的フィードバックを自動担保
  • マイルド化変換: ネガティブ表現を AI が建設的な表現に変換
  • 評価者インセンティブ: 評価行動自体を評価に加算

コンテンツ・ファンダメンタル

言語・トーン

完全日本語（Phase 1）、敬体（です・ます）基本。信頼・透明・落ち着き。過度な感情表現を避け、AI コーチング文脈では提案型で書く。

敬称・人称

「自分」「あなた」より役職・ロール名を優先（例: 評価対象者、マネージャー、被評価者）。絵文字は原則不使用。感嘆符も控えめに。

ビジュアル・ファンダメンタル

カラー

Sunbit Navy #1F2C69 — Primary Logo Gray #7A7B7F Success Warning Danger Info AI Violet

タイポグラフィ

  • 日本語 UI 本文: Noto Sans JP（400/500/700）
  • 欧文・数値: Inter（400/500/600/700）— tabular-nums 指定でダッシュボード用
  • モノスペース: JetBrains Mono（監査ログ・ID・コード）

レイアウト・形状

  • サイドバー 240px + ヘッダー 56px + コンテンツ最大幅 1440px
  • カード: 白背景 + 1px Slate-200 境界線 + 8px 角丸 + elevation-1
  • 角丸スケール: 4 / 6 / 8 / 12 / pill
  • シャドウ: 0 / 1 / 2 / 3 の 4 段階。Inner shadow は不使用

モーション・インタラクション

  • Easing: cubic-bezier(0.16, 1, 0.3, 1) / Duration: 150/200/300ms
  • Hover: bg-muted へ / Active: scale(0.98) + 4% 暗色化 / Focus: Primary 2px ring
  • バウンス・グラスモルフィズム・装飾的グラデーションは使わない

アイコノグラフィ

Lucide Icons（shadcn/ui デフォルト）を採用。ストローク 2px、24×24 viewBox。絵文字は UI 内で使用しない。Unicode の矢印（↗ ↑ ↓ ▲ ▼ ✓ ✗）のみ KPI 変化表示で許容。

カスタムアイコンセットが将来定義された場合、Lucide をベースに同じストローク幅 2px を踏襲する。

ロゴ

公式ワードマークを assets/logo.png（701×298, 透明 PNG）に配置。Navy 背景では filter: brightness(0) invert(1) で反転して白単色として使う。Mark 単体（logo-mark.svg）はサイドバー・アプリアイコン・favicon 用。
