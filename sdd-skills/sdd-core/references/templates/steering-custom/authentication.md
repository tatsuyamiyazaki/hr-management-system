## 認証フロー
```
1. ログインリクエスト（credentials）
2. 検証 → アクセストークン + リフレッシュトークン発行
3. リクエストにアクセストークン付与
4. 期限切れ → リフレッシュトークンで再発行
5. ログアウト → トークン失効
```
